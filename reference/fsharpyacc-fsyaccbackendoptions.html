FsyaccBackendOptions

Options to control the behavior of the fsyacc-compatible backend.

Record Fields

Record FieldDescription
InternalModule
Signature: bool

When set, the parser module will be generated with the 'internal' visibility modifier.
LexerInterpreterNamespace
Signature: string option

The namespace containing the lexer interpreter. If not set, a default value is used instead.
ModuleName
Signature: string option

The name to use for the generated parser module. If not set, a default value is used instead.
OpenDeclarations
Signature: string []

When generating the parser code, F# 'open' declarations are generated for these modules/namespace names (in addition to those emitted by default).
OutputPath
Signature: string

The output path. The generated code will be written to this file.
ParserInterpreterNamespace
Signature: string option

The namespace containing the lexer interpreter. If not set, a default value is used instead.
