LrParserActionSet

A non-empty set of LrParserActions representing the action(s) to take for a specific parser state.

Remove(action actionSet)
Signature: action:LrParserAction -> actionSet:LrParserActionSet -> LrParserActionSet option

Creates a new LrParserActionSet with the given action removed; returns None if the resulting action set would be empty. No exception is thrown if the set doesn't contain the given action.
