StartingProductions
Signature: NonterminalIdentifier list

The starting production(s) of the grammar. Only nonterminals can be specified, and all nonterminals specified as starting productions must also have explicit type declarations. At least one (1) nonterminal must be specified; a Specification is invalid if this field is empty.
